# https://3vorp.github.io

personal site that's supposed to plug all the stuff I've made but is really a slightly more formatted version of my github readme

## currently featuring
- gambling simulator
- barely functional mobile optimization
- sveltekit, scss, and typescript dependencies that are probably completely unnecessary (it's like 80% static content)
- a dependency chain so convoluted that if you breathe on it too hard it'll probably break
- text uwuifier (I'm sorry)
- super cool reactive navbar and footer components please check them out they took me so long

## running the site locally (why would you want to do this)
- install [nodejs](https://nodejs.org) 20+ and [pnpm](https://pnpm.io/)
- run `pnpm i` to install the dependencies
- run `pnpm dev` to start a local dev server
